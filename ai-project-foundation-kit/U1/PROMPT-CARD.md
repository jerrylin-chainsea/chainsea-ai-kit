# U1 · Prompt 卡

U1 主要靠手，不靠 AI。唯一一張卡：讓 AI「先讀、不要改」，認識你接手的專案。

## 卡 1 ｜ 讀專案（先看，不要改）

貼到 Claude Code 或 Codex：

```text
請先閱讀目前專案，不要修改任何檔案。

請回答：
1. 這個專案是做什麼的
2. 啟動方式是什麼
3. 主要檔案在哪裡
4. 如果我要修改首頁文字，可能要改哪裡
5. 我要怎麼驗收你沒有亂改
```

**驗收**：AI 給出摘要，而且 `git status` 乾淨（一個檔都沒被改）。

> 同一段話，在 Claude Code 和 Codex 都能用。差別是工具，核心是流程。
> 完整版與更多說明在 `prompts/01-read-project.md`。
