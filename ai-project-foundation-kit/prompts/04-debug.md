# 04 · Debug(用 ReAct 修錯:先想、再做、再看)

**什麼時候用:** 程式跑出來不對。例如 `python run.py` 的 `risk_level` 印出「嚴重」,但規格說只能是 `low / medium / high`。

**ReAct = Reason(先想)+ Act(做一個動作)+ Observe(看結果)。**

**第一步:先別改,先分析。複製這段:**

```
目前有一個錯誤:

python run.py 的輸出中,risk_level 不是 low、medium、high,
而是輸出了其他文字。

請先不要修改檔案。

請先分析:
1. 這個問題可能是什麼意思
2. 可能原因有哪些
3. 可能相關檔案
4. 你會怎麼確認問題
5. 建議修正方式
```

**第二步:看過分析、覺得合理,才允許它修。複製這段:**

```
同意,請照你的分析修正。
請只修改必要檔案。

修完後請告訴我:
1. 改了哪裡
2. 為什麼這樣改
3. 我要怎麼確認修好了
```

> 修好後再跑一次 `python run.py`,確認 risk_level 變成 low / medium / high。
