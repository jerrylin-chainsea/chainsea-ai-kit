# 01 ｜ 營運地圖模板

> 課1 用。把前面所有成果貼到三層地圖，標出資料來源、輸出、負責人。**只畫會用到的節點與箭頭**，不重做漂亮架構圖。

```
┌─────────────────────────────────────────────────────────┐
│ 對外層（品牌面對顧客）                                     │
│  ├─ M13 個人/品牌發布站（公開網址：__________）            │
│  ├─ 社群 / LINE OA（U8–9）                                │
│  └─ 客服回覆                                              │
├─────────────────────────────────────────────────────────┤
│ 內部營運層                                                │
│  ├─ 排班 / 薪資 / 成本 / 庫存試算表（U6）                  │
│  ├─ 營運系統前端（U7）                                    │
│  └─ 自動化流程 / 缺料叫料（U8）                            │
├─────────────────────────────────────────────────────────┤
│ 資料與決策層                                              │
│  ├─ 營運資料表（U5）                                      │
│  └─ M12 AI 決策助理（三角色 CrewAI）                       │
└─────────────────────────────────────────────────────────┘
```

| 節點 | 資料來源 | 輸出 | 負責人（組員） |
|---|---|---|---|
| 範例：M12 決策助理 | 營運資料表 | 決策摘要 | ____ |
| | | | |
