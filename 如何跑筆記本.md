# 如何跑筆記本(.ipynb)— 第一次卡住看這頁

筆記本要能跑,需要一個有 `ipykernel` 的 Python 環境。在 Codespaces 裡通常已經幫你裝好了;若卡住,照下面做。

## 跑筆記本的三步

1. 打開 `.ipynb`,點右上角 **Select Kernel(選取核心)**。
2. 選 **Python 3**(或顯示「Python 3 (課程)」的那個)。
   - ⚠️ **不要選名字裡有 `.venv` 的** — 那個常常沒裝 ipykernel,就會跳錯。
3. 從上往下,一格一格按 **▶**。

## 如果它叫你裝 ipykernel,而且「Install」按鈕失敗

別用那個按鈕。改開 **終端機(Terminal)**,貼這行、按 Enter:

```bash
python3 -m pip install ipykernel
```

裝完回筆記本,再 **Select Kernel** 選一次 **Python 3**。

## 各筆記本需要什麼

| 筆記本 | 需要 |
|---|---|
| `m14-auto-ops-starter/M14_自動經營筆記本.ipynb` | 只要 `ipykernel`(其他都是內建的) |
| `m12-crew-starter/M12_課1_教學筆記本.ipynb` | `ipykernel` + `pip install -r m12-crew-starter/requirements.txt`(devcontainer 會自動裝) |

## 完全不想用筆記本?直接跑也行

```bash
cd m14-auto-ops-starter && python3 run.py     # M14 自動經營,看日誌
cd m12-crew-starter     && python3 run.py     # M12 三角色(需金鑰)
```

> 重點:看得到輸出就是成功。卡在環境就用上面終端機那行,別跟 `.venv` 較勁。
